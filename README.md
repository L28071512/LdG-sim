# LdG-sim
An implementation of Landau-de Gennes's model of nematic liquid crystal simulation  

## Prerequisites
- Python 3
- Numpy
- Matplotlib

## Download
```shell
git clone https://github.com/tings0802/LdG-sim.git
```
